https://leetcode.com/problems/gas-station/

能到下一站的話 必定是 ${到前一站時之剩餘油量} + ${前一站所加之油量} - ${前一站這一站所耗之油量} >= 0
第一步是把兩個列表合併成到此這所加之油到下一站剩餘量, 意即
```python
transit = [x - y for x,y in zip(gas, cost)]
```
然後計算從哪一站開始可以連續 num = len(transit) 站都大於 0

結束條件:
1. 連續 num 站之剩餘油量皆大於或等於 0: 回傳開始跑的那一站
2. 剩餘油量小於 0 並且已經跑一輪: 回傳 -1
